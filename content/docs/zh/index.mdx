## 概览

Dockerman 是一个基于 Tauri 和 Rust 构建的现代轻量级 Docker 管理界面，专注于简洁和性能。

## 功能特性

- 容器生命周期管理（创建、启动、停止、重启、删除）
- 镜像管理（拉取、构建、删除）
- 实时资源监控（CPU、内存、网络）
- 内置终端，直接访问运行中的容器
- 跨平台支持（macOS、Windows、Linux）
