## 安装

从[官方网站](https://dockerman.app/zh/download)下载 Dockerman。

### 系统要求

- macOS 12+ / Windows 10+ / Linux (Ubuntu 20.04+)
- 已安装并运行 Docker Engine

### 安装步骤

1. 下载适用于您平台的安装包
2. 运行安装程序
3. 启动 Dockerman
4. Dockerman 将自动检测您的 Docker Engine

## 初次使用

启动 Dockerman 后，您将看到仪表板，其中展示了 Docker 环境的概览信息。
