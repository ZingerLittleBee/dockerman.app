## インストール

Dockerman は[公式ウェブサイト](https://dockerman.app/ja/download)からダウンロードできます。

### システム要件

- macOS 12+ / Windows 10+ / Linux（Ubuntu 20.04+）
- Docker Engine がインストールされ、実行中であること

### 手順

1. プラットフォーム用のインストーラーをダウンロード
2. インストーラーを実行
3. Dockerman を起動
4. Dockerman は自動的に Docker Engine を検出します

## 最初のステップ

Dockerman を起動すると、Docker 環境の概要が表示されるダッシュボードが表示されます。
