## 概要

Dockerman は、Tauri と Rust で構築されたモダンで軽量な Docker 管理インターフェースです。シンプルさとパフォーマンスに焦点を当てています。

## 機能

- コンテナライフサイクル管理（作成、開始、停止、再起動、削除）
- プル、ビルド、削除操作を含むイメージ管理
- CPU、メモリ、ネットワーク統計のリアルタイムリソース監視
- 実行中のコンテナへの組み込みターミナルアクセス
- クロスプラットフォーム対応（macOS、Windows、Linux）
