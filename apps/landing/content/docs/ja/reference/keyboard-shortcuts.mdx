## グローバルショートカット

| ショートカット | アクション |
|-------------|---------|
| `Cmd/Ctrl + ;` | コマンドパレットを開く |
| `Cmd/Ctrl + Shift + T` | 新しいターミナルを開く |
| `Cmd/Ctrl + ,` | 設定を開く |

## コンテナビュー

| ショートカット | アクション |
|-------------|---------|
| `R` | コンテナを再起動 |
| `S` | コンテナを停止 |
| `K` | コンテナを削除 |
| `L` | ログを表示 |

## ナビゲーション

| ショートカット | アクション |
|-------------|---------|
| `Cmd/Ctrl + 1` | ダッシュボード |
| `Cmd/Ctrl + 2` | コンテナ |
| `Cmd/Ctrl + 3` | イメージ |
