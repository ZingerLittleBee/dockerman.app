---
title: Keyboard Shortcuts
description: Every keyboard shortcut in Dockerman, grouped by surface — global, lists, logs, terminal, and Kubernetes.
---

Every keyboard shortcut in Dockerman is grouped by surface. Most shortcuts are customizable in [Settings → Shortcut settings](/en/docs/reference/settings#shortcut-settings).

## Global

| Shortcut | Action |
| --- | --- |
| Cmd/Ctrl + K | Open Spotlight |
| Cmd/Ctrl + , | Open Settings |
| Cmd/Ctrl + Shift + L | Toggle theme |
| Cmd/Ctrl + Shift + H | Open Host switcher |

## Navigation

| Shortcut | Action |
| --- | --- |
| Up / Down | Move selection in the current list |
| Enter | Open the selected row |
| Cmd/Ctrl + Click | Toggle multi-select |
| Shift + Click | Select a range |

## Container actions

| Shortcut | Action |
| --- | --- |
| S | Start |
| X | Stop |
| R | Restart |
| P | Pause or unpause |
| L | Logs |
| T | Terminal |
| I | Inspect |
| Delete | Remove after confirmation |

## Log viewer

| Shortcut | Action |
| --- | --- |
| / | Open search |
| n | Next match |
| N | Previous match |
| g | Jump to the start |
| G | Jump to the latest line |
| P | Previous match, same as Shift + n |
| Esc | Close search or clear highlights |

## Terminal

| Shortcut | Action |
| --- | --- |
| Ctrl/Cmd + Shift + C | Copy selection |
| Ctrl/Cmd + Shift + V | Paste |
| Ctrl/Cmd + Shift + K | Clear the terminal |
| Ctrl + C | Send SIGINT to the process |
| Ctrl + D | Send EOF |

## Kubernetes

| Shortcut | Action |
| --- | --- |
| N | Cycle namespace |
| D | Debug on a Pod |
| Y | Edit YAML |

## Customize

Change any shortcut in [Settings → Shortcut settings](/en/docs/reference/settings#shortcut-settings).
