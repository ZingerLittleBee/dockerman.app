## Instalación

Descarga Dockerman desde el [sitio web oficial](https://dockerman.app/es/download).

## Requisitos del Sistema

- macOS 12+ / Windows 10+ / Linux (Ubuntu 20.04+)
- Docker Engine instalado y en ejecución

## Pasos

1. Descarga el instalador para tu plataforma
2. Ejecuta el instalador
3. Lanza Dockerman
4. Dockerman detectará automáticamente tu Docker Engine

## Primeros Pasos

Después de lanzar Dockerman, verás el panel con un resumen de tu entorno Docker.
