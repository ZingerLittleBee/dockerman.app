## Resumen

Dockerman soporta las principales distribuciones Linux.

## Distribuciones Soportadas

- Ubuntu 20.04+
- Debian 11+
- Fedora 35+
- Arch Linux

## Instalación

Descarga el formato de paquete para tu distribución:
- `.deb` (Debian/Ubuntu)
- `.rpm` (Fedora/RHEL)
- `.AppImage` (Universal)
