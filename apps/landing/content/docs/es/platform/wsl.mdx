## Resumen

Dockerman puede conectarse con Docker dentro de WSL2 en Windows.

## Configuración

1. Asegúrate de tener WSL2 instalado
2. Instala Docker en tu distribución WSL2
3. Selecciona el socket WSL en la configuración de Dockerman

## Ruta del Socket WSL2

Por defecto: `\\wsl$\<distro>\var\run\docker.sock`
