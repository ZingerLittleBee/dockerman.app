## Resumen

Navega, edita y descarga el sistema de archivos dentro de contenedores.

## Operaciones de Archivos

- **Navegar**: Moverse por directorios
- **Ver**: Previsualizar archivos de texto
- **Editar**: Editar y guardar archivos
- **Descargar**: Descargar archivos
- **Subir**: Subir archivos al contenedor

## Formatos Soportados

- Archivos de texto
- Imágenes
- JSON, XML, YAML
