## Resumen

Dockerman proporciona una interfaz intuitiva para gestionar el ciclo de vida de contenedores.

## Operaciones de Contenedores

- **Iniciar**: Iniciar un contenedor detenido
- **Detener**: Detener un contenedor en ejecución
- **Reiniciar**: Reiniciar un contenedor
- **Eliminar**: Eliminar un contenedor

## Detalles del Contenedor

Haz clic en un contenedor para ver información detallada:
- Mapeo de puertos
- Variables de entorno
- Montajes de volúmenes
- Logs
