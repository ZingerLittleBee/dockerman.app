## Resumen

Crea, gestiona y elimina redes Docker.

## Operaciones de Redes

- **Crear**: Crear una nueva red
- **Eliminar**: Eliminar una red
- **Inspeccionar**: Ver detalles de la red

## Tipos de Redes

- bridge
- host
- none
- overlay
