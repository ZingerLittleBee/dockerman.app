## Resumen

Dockerman proporciona soporte completo para proyectos Docker Compose.

## Operaciones de Compose

- **Up**: Iniciar proyecto Compose
- **Down**: Detener proyecto Compose
- **Pull**: Pull de imágenes
- **Build**: Construir servicios

## Archivos Compose

Puedes arrastrar y soltar archivos `docker-compose.yml` directamente en Dockerman.
