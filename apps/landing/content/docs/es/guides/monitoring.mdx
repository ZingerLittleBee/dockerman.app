## Resumen

Monitorea el uso de recursos de contenedores en tiempo real.

## Métricas

- **Uso de CPU**: Estadísticas de CPU en tiempo real
- **Uso de Memoria**: Seguimiento de uso de memoria
- **I/O de Red**: Tráfico de red
- **I/O de Disco**: Estadísticas de almacenamiento

Gráficos de series temporales para ver datos históricos.
