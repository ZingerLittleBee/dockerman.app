## Resumen

Dockerman es gratuito para usar. Las funciones de host remoto requieren una licencia.

## Versión Gratuita

- Gestión de host Docker local
- Todas las funciones de contenedores e imágenes
- Panel y monitoreo

## Versión con Licencia

- Múltiples hosts remotos
- Conexión segura vía SSH
- Actualizaciones de por vida

## Activación de Licencia

1. Abre Configuración → Licencia
2. Ingresa tu clave de licencia
3. Haz clic en "Activar"
