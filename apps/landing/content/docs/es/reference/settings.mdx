## Resumen

Personaliza las opciones de configuración de Dockerman.

## Configuración General

- **Idioma**: Selecciona el idioma de la UI
- **Tema**: Modo claro/oscuro
- **Al Inicio**: Iniciar Dockerman al arrancar el sistema

## Configuración Docker

- **Ruta del Socket**: Socket Docker personalizado
- **Host Remoto**: Conexión Docker remota vía SSH

## Privacidad

- **Analíticas**: Habilitar/deshabilitar analíticas de uso
