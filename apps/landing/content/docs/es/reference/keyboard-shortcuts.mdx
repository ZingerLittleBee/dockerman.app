## Atajos Globales

| Atajo | Acción |
|-------|--------|
| `Cmd/Ctrl + ;` | Abrir paleta de comandos |
| `Cmd/Ctrl + Shift + T` | Abrir nueva terminal |
| `Cmd/Ctrl + ,` | Abrir configuración |

## Vista de Contenedores

| Atajo | Acción |
|-------|--------|
| `R` | Reiniciar contenedor |
| `S` | Detener contenedor |
| `K` | Eliminar contenedor |
| `L` | Ver logs |

## Navegación

| Atajo | Acción |
|-------|--------|
| `Cmd/Ctrl + 1` | Panel |
| `Cmd/Ctrl + 2` | Contenedores |
| `Cmd/Ctrl + 3` | Imágenes |
