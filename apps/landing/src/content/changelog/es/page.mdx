## Soporte del Runtime Podman

Esta versión introduce soporte de primera clase para el runtime Podman junto con Docker, con detección automática de runtime, selector de preferencia por host y un sistema extensible de capabilities para el gating de funciones.

### ✨ Funcionalidades

- 🐙 <Bold>Soporte Podman:</Bold> soporte de primera clase para el runtime Podman junto con Docker
  - 🔍 Detección de runtime: detección automática de Docker, Podman o runtimes compatibles vía API
  - ⚙️ Preferencia de runtime: selector por host y global (Auto / Docker / Podman)
  - 🔌 Resolución de socket: descubrimiento automático de sockets Podman rootless/rootful en Linux y macOS
  - 🌐 Probing SSH remoto: resolución de socket consciente del runtime para hosts SSH remotos
  - 🏷️ Badge de runtime: el switcher de host en la barra lateral muestra el runtime detectado con badge a color
  - 📊 Integración con el dashboard: el tipo de runtime se muestra en la tarjeta de información del sistema
  - 🔧 Feedback de conexión: runtime detectado mostrado en la salida del terminal al conectar
  - 🚫 Gate de Compose: plantillas de stack y acciones Compose ocultas automáticamente al conectar a Podman
  - 🔎 Conciencia en Spotlight: la paleta de comandos omite datos de Compose cuando se conecta a Podman
  - 🧩 Capa de capabilities: sistema extensible de capabilities de runtime para gating futuro

### 🔧 Mejoras

- 🔄 <Bold>Cambio de runtime:</Bold> la página de plantillas se actualiza dinámicamente al alternar entre hosts Docker y Podman
- 🛡️ <Bold>Aislamiento de Spotlight:</Bold> los chequeos de runtime de Spotlight usan comandos Tauri directos sin efectos en el store

## Soporte de Kubernetes y Túneles en la Nube

Esta versión principal introduce gestión completa de Kubernetes, túneles Cloudflared para exposición a internet con un clic, detección automática de actualizaciones de imágenes con rollback, un panel de información de contenedores renovado y dos nuevos idiomas.

### ✨ Características

- ☸️ <Bold>Soporte de Kubernetes:</Bold> Gestión completa de Kubernetes junto con Docker
  - 🚀 Gestión de Clústeres: Conectar vía kubeconfig o crear clústeres locales con k3d
  - 📦 Cargas de Trabajo: Gestionar Deployments, StatefulSets, DaemonSets, Jobs, CronJobs, ReplicaSets y Pods
  - 🌐 Redes: Services, Ingresses, Endpoints y NetworkPolicies
  - 🔧 Configuración y Almacenamiento: ConfigMaps, Secrets, PersistentVolumeClaims y StorageClasses
  - 🔐 RBAC: Roles, ClusterRoles, RoleBindings, ClusterRoleBindings y ServiceAccounts
  - 🧩 Recursos Personalizados: Explorar CRDs y gestionar sus instancias
  - 📝 Editor YAML: Editar cualquier recurso con soporte de enlaces profundos
  - ⎈ Soporte de Helm: Explorar releases, agregar repositorios, buscar charts e instalar
  - 🔗 Reenvío de Puertos: Reenviar puertos para Pods, Services y Deployments
  - 🌍 DNS Automático: Registro DNS local automático para servicios reenviados
  - 🛠️ Asistente de Depuración: Página de diagnóstico para problemas de Pods y clústeres
- ☁️ <Bold>Túneles Cloudflared:</Bold> Exponer puertos de contenedores locales a internet con un clic
  - 🔘 Botón de túnel con un clic en la tabla de puertos del contenedor
  - 📋 Lista dedicada de túneles en la barra lateral con operaciones por lotes
  - 🔄 Limpieza automática al detener/destruir contenedores
  - 💾 Recuperación persistente de túneles después de fallos
- 🔄 <Bold>Detección de Actualización de Imágenes:</Bold> Detectar automáticamente actualizaciones disponibles
  - 🏷️ Insignias de actualización en la lista de contenedores con comparación de digest
  - 🖱️ Actualización con un clic con pull de imagen y recreación de contenedor
  - 🔁 Rollback con un clic desde el menú de acciones
  - ⚡ Verificaciones automáticas al cargar, refrescar y cambiar de host
- 📊 <Bold>Panel de Información de Contenedor:</Bold> Página de información renovada con diseño de panel
  - 🎯 Variables de entorno, montajes y límites de recursos en paneles dedicados
  - ❤️ Estado de healthcheck junto a CPU y memoria
- 🌐 <Bold>Japonés y Español:</Bold> Localización completa para ja y es (4 idiomas en total)
- 🔌 <Bold>Columna de Puertos:</Bold> Nueva columna de puertos con menú desplegable
- 📋 <Bold>Visor de Logs Unificado:</Bold> LogViewer compartido para logs de Docker y Kubernetes
- 📜 <Bold>Desplazamiento Horizontal:</Bold> Todas las tablas principales soportan desplazamiento horizontal
- 🧭 <Bold>Navegación K8s con Migas:</Bold> Enlaces de migas de pan clicables para la jerarquía de recursos

### 🔧 Mejoras

- 📌 <Bold>Filas Expandidas Fijas:</Bold> Las filas expandidas permanecen fijas al desplazarse
- 📏 <Bold>Persistencia de Ancho de Columnas:</Bold> Los anchos redimensionables se recuerdan entre sesiones
- 🔍 <Bold>Mostrar Todo al Buscar:</Bold> La búsqueda incluye automáticamente contenedores detenidos

## Primera Versión Oficial

Nos complace anunciar la primera versión oficial de Dockerman: una UI moderna y ligera para gestión Docker que brinda potente rendimiento y simplicidad para la gestión de contenedores.

### Características Principales

- <Bold>Vista General del Panel:</Bold> Acceso instantáneo a estadísticas de contenedores e imágenes, con monitoreo en tiempo real del uso de recursos y actualizaciones de estado de contenedores.
- <Bold>Gestión de Contenedores:</Bold> Control comprehensivo de contenedores con capacidades de inspección detallada, acceso integrado a terminal y monitoreo de recursos en tiempo real.
- <Bold>Gestión de Imágenes:</Bold> Manejo eficiente de imágenes con análisis detallado de tamaño, información de etiquetas y seguimiento de uso.
- <Bold>Monitoreo de Recursos:</Bold> Visualización en tiempo real del uso de CPU, consumo de memoria, estadísticas de red y rendimiento de E/S de bloques.
- <Bold>Integración del Sistema:</Bold> Integración nativa con el demonio Docker proporcionando rendimiento y confiabilidad superiores.

### Puntos Destacados

- <Bold>Rendimiento Potente:</Bold> Construido con Tauri para velocidad excepcional y mínima huella de recursos.
- <Bold>Integración de Terminal:</Bold> Acceso integrado a terminal para interacción directa con contenedores.
- <Bold>Gestión de Procesos:</Bold> Visualización y monitoreo de lista de procesos en tiempo real.
- <Bold>Visor de Logs:</Bold> Capacidades avanzadas de visualización de logs con actualizaciones en tiempo real.
- <Bold>Enfoque en Seguridad:</Bold> Operación solo local asegurando que los datos de tus contenedores permanezcan seguros.

</ChangelogEntry>
