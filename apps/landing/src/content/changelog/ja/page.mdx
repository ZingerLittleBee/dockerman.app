## コンテナのバックアップと復元

このリリースでは、進捗追跡、プレビュー、Spotlight クイックアクセスコマンドを備えた、完全なコンテナバックアップと復元機能が導入されました。

### ✨ 新機能

- 💾 <Bold>コンテナバックアップ:</Bold> 進捗追跡付きの完全なコンテナバックアップと復元
- 📦 <Bold>すべてをアーカイブ:</Bold> コンテナ設定、ファイルシステム、ボリュームを単一のアーカイブにバックアップ
- 🔄 <Bold>スマート復元:</Bold> プレビュー、ポート編集、名前の競合解決、イメージの再タグ付けを備えたバックアップからの復元
- 🔍 <Bold>バックアッププレビュー:</Bold> 復元前に設定の詳細を表示
- ⚡ <Bold>ストリーミング進捗:</Bold> キャンセルサポート付きのリアルタイム進捗追跡
- 🔎 <Bold>Spotlight コマンド:</Bold> バックアップと復元操作へのクイックアクセス

</ChangelogEntry>

## Docker Hub ブラウザとセキュリティスキャン

このリリースでは、コンテナの一時停止/再開、コミット、クローン操作、イメージのレジストリへのプッシュ、組み込み Docker Hub ブラウザ、Trivy セキュリティスキャンが追加されました。

### ✨ 新機能

- ⏸️ <Bold>コンテナの一時停止/再開:</Bold> 3状態メニュー対応で実行中のコンテナを一時停止および再開
- 📦 <Bold>コンテナコミット:</Bold> コミットダイアログでコンテナ状態を新しいイメージとして保存
- 🐑 <Bold>コンテナクローン:</Bold> 既存のコンテナ設定を作成ダイアログにクローン
- 🚀 <Bold>イメージプッシュ:</Bold> ストリーミング進捗と認証情報管理付きでイメージをレジストリにプッシュ
- 🔍 <Bold>Docker Hub ブラウザ:</Bold> アプリ内でイメージ検索、詳細閲覧、README、タグを直接確認
- 🛡️ <Bold>Trivy セキュリティスキャン:</Bold> Trivy のインストール、イメージの脆弱性スキャン、結果のフィルタリングとレビュー

</ChangelogEntry>

<ChangelogEntry version="v4.0.0" date="2026年2月17日">
## コマンドパレットとシステムトレイ

このリリースでは、リソースへの即時アクセスのためのグローバルコマンドパレット、リアルタイム統計付きシステムトレイ統合、macOS Dock アイコンの自動非表示が導入されました。

### ✨ 新機能

- 🔍 <Bold>コマンドパレット:</Bold> グローバルコマンドパレット（Cmd+; / Ctrl+;）で、コンテナ、イメージ、Compose プロジェクト、ネットワーク、ボリュームへのクイックアクセスと即時操作
- 🖥️ <Bold>システムトレイ:</Bold> メニューバーのトレイアイコンで、接続状態、リソース統計、クイックアクション（ウィンドウの表示/非表示、スポットライト、設定、終了）を表示
- 📊 <Bold>メニューバー統計:</Bold> macOS メニューバーに直接リアルタイム CPU とメモリ統計を表示（設定で構成可能）
- 🙈 <Bold>Dock アイコンの自動非表示:</Bold> macOS でメインウィンドウを閉じると Dock アイコンが自動的に非表示
- ⌨️ <Bold>カスタムショートカット:</Bold> コマンドパレット用の設定可能なグローバルキーボードショートカット
- 🌐 <Bold>i18n:</Bold> コマンドパレットとトレイ設定の完全な日本語・英語ローカリゼーション

</ChangelogEntry>

## Docker イベントとコンテナファイルブラウザ

このリリースでは、リアルタイム Docker イベント監視と包括的な Docker コンテナファイルブラウザが導入されました。

### ✨ 新機能

- <Bold>Docker イベント監視:</Bold> 包括的なイベントフィルタリング、検索、イベントデータ検査用の JSON 詳細ダイアログを備えたリアルタイム Docker イベントリスニング
- <Bold>コンテナファイルブラウザ:</Bold> Docker コンテナ用のファイルシステムエクスプローラを実装：
  - 🌳 動的読み込みを備えたファイルツリーナビゲーション
  - 📂 フォルダのアップロードとダウンロード対応（tar ベース）
  - 📄 モード保持付きの単一ファイルのアップロードとダウンロード
  - 🔗 ファイルリストのシンボリックリンク対応
  - 📄 大きなディレクトリのページネーション
  - 📋 ファイル/フォルダ操作のコンテキストメニュー（ダウンロード、パスのコピー、プロパティの表示）

</ChangelogEntry>

<ChangelogEntry version="v1.0.0" date="2025年2月4日">
## 最初の正式リリース

コンテナ管理に強力なパフォーマンスとシンプルさをもたらす、モダンで軽量な Docker 管理 UI「Dockerman」の最初の正式リリースを発表できることを嬉しく思います。

### コア機能

- <Bold>ダッシュボード概要:</Bold> コンテナとイメージの統計への即時アクセス、リソース使用状況とコンテナ状態更新のリアルタイム監視。
- <Bold>コンテナ管理:</Bold> 詳細な検査機能、統合ターミナルアクセス、リアルタイムリソース監視を備えた包括的なコンテナ制御。
- <Bold>イメージ管理:</Bold> 詳細なサイズ分析、タグ情報、使用状況追跡を備えた効率的なイメージ処理。
- <Bold>リソース監視:</Bold> CPU 使用率、メモリ消費、ネットワーク統計、ブロック I/O パフォーマンスのリアルタイム可視化。
- <Bold>システム統合:</Bold> Docker デーモンとのネイティブ統合により、優れたパフォーマンスと信頼性を提供。

### 主なハイライト

- <Bold>強力なパフォーマンス:</Bold> Tauri で構築され、卓越した速度と最小限のリソースフットプリントを実現。
- <Bold>ターミナル統合:</Bold> コンテナとの直接インタラクションのための組み込みターミナルアクセス。
- <Bold>プロセス管理:</Bold> リアルタイムのプロセスリスト表示と監視。
- <Bold>ログビューア:</Bold> リアルタイム更新を備えた高度なログ表示機能。
- <Bold>セキュリティ重視:</Bold> コンテナデータのセキュリティを確保するローカルオンリー操作。

</ChangelogEntry>
